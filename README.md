# Yufei Wang — Research & Creative Portfolio

A bilingual portfolio presenting Yufei Wang's research, interactive media, cultural communication, AIGC, tourism innovation, and creative technology projects.

## Highlights

- English and Chinese language switch with persistent preference
- Research projects, selected creative projects, experience, awards, and snapshots
- Interactive work including Pixel Hainan, a Unity mini game, learning games, and digital storytelling
- Responsive layout for desktop and mobile browsers

## Local development

```bash
npm install
npm run dev
```

Create a production build with:

```bash
npm run build
```
